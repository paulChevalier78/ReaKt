Features & Interface

3D Digital Twin

Animated 3D bioreactor (Plotly) showing volume, RPM, aeration bubbles, and color changes based on biomass.

Real-time Monitoring

Live KPIs (Penicillin, Biomass, Volume) and control variables.

Interactive Playback

"DVR" mode to replay batches and inspect MPC decisions.

Process Charts

Plots of sugar, air, acid, and base against biological outputs.

Hardware-Aware

Runs on CPU/GPU with efficient memory management.

Simple Setup

Train, save the model + scalers, and run the Streamlit app with your data.
